FireFly 总结

FireFly 命令

  1. hff 会直接显示命令， 效果等同于 hff -?
  2. hff create 文件名 文件名 文件名
  3. hff submit 文件名 文件名 文件名
  4. hff edit 文件名 文件名 文件名
  5. hff list-out 目录，查看是否有 edit，但没有submit的文件
  6. hff list-hijacked 目录，查看是否有强行非法修改的文件
  7. hff list-nwf 目录，查看是否有没有添加到Firefly管理的文件
  8. hff diffs 文件名 查看版本区别

FireFly 问题

  • firefly 提交或签出 No changeset found 错误
    在Firefly Command Line 中，进入工作区目录，输入：hff check -repair , 即可修复变更集
