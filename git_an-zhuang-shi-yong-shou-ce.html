版本库手册（纯GitHub版）

打开解压后的github(GitHub.exe)

样式图——略

  1. 创建新的本地版本库， create（空文件夹拖进来也行)

  2. 配置信息

  3. 打开， 点击 tools-> settings

  4. 配置远程版本库信息
    填写 用户名@IP:相对路径
    用户名 -- 我的版本库都是rbcmb
    IP -- 我的电脑IP 99.6.145.70
    路径 -- tools 从tools可以拿到其他几个

  5. 完成后，同步 sync

版本库手册（git命令行 + GitHub版）

  1. 在git bash或cmd里
    克隆到当前目录 

    git clone rbcmb@99.6.145.70:rbcmb

    克隆到指定目录

    git clone rbcmb@99.6.145.70:rbcmb  path

  2. 将文件夹拖进github

基础使用手册

注意

  1. 同步之前，本地变更集需要先提交完毕，再同步，否则文件容易混乱。
  2. 可以使用命令行、Git Gui、GitHub等多种方式查看log。推荐比较全面的gui工具 Tortoise Git， 虽然用到的时候不多。

本地配置

进行操作前的配置，也是安装后的配置：

git config --global user.name 你的名字 省略 --global 也行

git config --global user.email "你的邮件地址"

等同于 tortoise git 右键 Settings-Git-Edit local(global) .git/config

等同于GitHub里 create键旁边的 tools->options->configure git
