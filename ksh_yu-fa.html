% shell 语法
% 杨秀隆
% 2014-06-11

数学运算

使用的是 (( 运算式 ))

示例 :

(( bch_id += 1 )) 不支持 ++ 
(( next = pre + 1 )) 不能用 next=((pre+1))

或是 let var=运算式

示例:

let new_rank=$old_rank-1

进制情况 value too great for base :

let new_rank=10#$old_rank-1  十进制

分支条件

shell 或者说 ksh 的else if 使用 elif 写法， 简洁易用的写法

示例:

if [[ x -eq "3456" ]] then
    echo "3456"
elif [[ x -eq "4567" ]] then
    echo "4567"
elif [[ x -eq "1234" ]] then
    echo "1234"
else
    echo "none"
fi
