quick union 快速合并

数据结构

  1. 大小为N 的整型数组
  2. 解释: id[i] 是 i 的父结点。
  3. i 的根结点就是 id[id[id[...id[i]...]]]

java 实现重点

public boolean connected(int p, int q)
{
    return root(p) == root(q)
}

private int root(int i)
{
    while (i != id[i]) i = id[i];
    return i;
}

public void union(int p, int q)
{
    int i = root(p);
    int j = root(q);
    id[i] = j;
}

算法分析

成本分析:

  1. 初始化: N
  2. 合并: N
  3. 查找: N

缺陷:

  1. 生成树太高、太深
  2. 查找的成本也高。
