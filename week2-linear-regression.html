一元线性回归

  1. SSE -- sum of squared errors
  2. RMSE -- Root-Mean-Square Error
  3. R^2 -- compare the best model to a "baseline" model R^2 = 1 - SSE/SST
  4. baseline -- avg, not use any variables, predict the avg value of the dependent variable.

  5. SST total sum of squares

  6. R^2 meaning

    1. using a model 的效果
    2. = 0 则无效果
    3. = 1 则是完美

多元线性回归

  1. 多元可以对一元进行优化？ 跟没说差不多
  2. 添加更多元变量能提高 model
  3. diminishing returns as more variables are added

不要使用过多元（维）变量 1. 每个变量都会多算很多数据 2. 建模时 R^2 很高， 但对未知数据的预测反而不准， 即为 过拟合
