profile设置

Date 日 04 五月 2014

ksh profile常用配置

  1. stty erase ^H
    按Backspace键可以向后删除字符，
    输入stty erase ctrl+v ctrl+h

  2. set -o vi
    让输入行工作于vi模式,则可以使用vi专用指令了
    可以使用 set -o emacs， 则是使用emacs指令

  3. TERM=vt100
    这样在异地登录的用户方可使用vi
    各人各用， 这条不要用，除非我不用的环境……

  4. PS1=/usr/ucb/whoami"@"hostname:${ORACLE_SID}"$";export PS1
    包含登陆用户、主机名、数据库SID的提示符

  5. EDITOR=vi;export EDITOR
    不设置则无法crontab -e crontab

  6. HISTSIZE=200;export HISTSIZE
    保存的历史命令数

  7. LANG=zh_CN.GBK;export LANG;
    让系统显示中文信息, 已有

  8. stty -istrip cs8
    将所有的输入截取成8bits的资料，因为中文的资料都是8bits的
    未使用过

  9. LC_CTYPE=zh_CN.GBK;export LC_CTYPE
    让系统可以使用中文 未使用过

  10. alias history='fc -l'
    显示历史命令
    可能还是原命令短一点，但是history好记

  11. alias r='fc -e -' 重复执行最后一条命令, 慎用
