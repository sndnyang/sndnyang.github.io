源代码组织

源文件

.h .c .cpp .hpp

库文件

  • 动态库
    .a(UNIX) / .dll(Windows)
    优点:

    1. 执行过程中装载
    2. 程序较小
    3. 共享动态库 注意点: 动态库需要一同发布

  • 静态库
    .so(UNIX) / .lib(Windows) 优点：

    1. 执行时不需要静态库 注意点：
    2. 链接时，库中数据加入
    3. 占用空间、资源多

静态变量 static

特点: 1. 分配在全局数据区 2. 在声明时初始化一次 3. 不显式初始化，则为0

外部变量 extern

匈牙利命名法

组成： 

作用域前缀 _类型简称[ _基本名称][ _限定词]

m : 类的成员变量 sm : 类的静态成员变量 g : 全局变量 s : 静态变量

参数

  1. 某个参数（指针或引用）功能上不会在函数中修改时， 必须定义为const
  2. 尽量用引用传参
  3. 函数对参数进行合法性检查
