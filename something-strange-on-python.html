不知道这个是什么¶

AttributeError: MSVCCompiler instance has no attribute 'compiler_so'

error: Unable to find vcvarsall.bat

似乎需要在 windows下指定 mingw的环境，所以

python setup.py build -c mingw32

还需要安装mingw， 不然什么 gcc，及头文件都没有。
