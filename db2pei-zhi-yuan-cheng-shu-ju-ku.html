db2配置远程数据库

配置远程数据库

1. 执行db2 catalog tcpip node 节点名 remote xx.xx.xx.xx server 50000

  1. 执行db2 list node directory show detail，显示可用的节点，确认NODECRM已加入成功

  2. db2 catalog db xxxx as 别名 at node 节点名

  3. 执行InitDbKey,输入数据库用户名和密码（见上）

  4. 执行db2 connect to xxxx GetDbKey xxxx,验证一下是否能正常连接数据库

  5. 进到sqldb2.bnd文件所在的目录(一般在~/bnd/ORCHID下,执行 db2 bind sqldb2.bnd
