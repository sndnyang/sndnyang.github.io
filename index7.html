profile设置

ksh profile常用配置

  1. stty erase ^H
    按Backspace键可以向后删除字符，
    输入stty erase ctrl+v ctrl+h

  2. set -o vi
    让输入行工作于vi模式,则可以使用vi专用指令了
    可以使用 set -o emacs， 则是使用emacs指令

  3. TERM=vt100
    这样在异地登录的用户方可使用vi
    各人各用， 这条不要用，除非我不用的环境……

  4. PS1=/usr/ucb/whoami"@"hostname:${ORACLE_SID}"$";export PS1
    包含登陆用户、主机名、数据库SID的提示符

  5. EDITOR=vi;export EDITOR
    不设置则无法crontab -e crontab

  6. HISTSIZE=200;export HISTSIZE ...

if else elif

shell 或者说 ksh 的else if 使用 elif 写法， 简洁易用的写法

    echo "3456"
elif [[ x -eq "4567" ]] then
    echo "4567"

    echo "1234"
else

markdown_syntax

MarkDown 笔记

小细节记录

  1. 非MarkDown的部分，可直接写HTML
  2. < div >, < table>, < pre>, < p> 前后要加空行，不能使用空格等来缩进
  3. HTML区块（block)标签内的MarkDown不做语法处理。
  4. 区段（inline)标签则随意，比如 < span>, < cite>, < del>

一、 区块元素

1. 段落和换行

段落前后有一个以上空行（一般自然就是前后各空一行），不能用空格或制表符来缩进。

2. 标题

类Setext 底线形式，只有＝（最高阶标题）和－（第二阶标题） 类Atx 形式，使用＃，1至6个，对应1阶到6阶

可以使用闭合式标题，即前后配对，出于美观考虑
